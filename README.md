# CountDown Light Switch

A modern Android application built with Jetpack Compose that allows users to automate light switching using countdown timers and scheduled actions.

## Features

- Countdown-based light control
- Modern Material Design UI
- Background task execution
- Local data persistence
- Notifications
- Dark Mode support
